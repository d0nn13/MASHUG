SPACE INVADERS
==============
A well known arcade game...

Requirements :
--------------
SDL2, SDL2_image, SDL2_ttf :

```sh
brew install sdl2 sdl2_image, sdl2_ttf -v
```
Building :
----------
```
make
```
Launching :
-----------
```
./bin/spaceinvaders-dbg
```

#####You have to launch the binary from the root of the repo (see *#2*)
