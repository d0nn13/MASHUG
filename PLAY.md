MASHUG
======
Multiple Arcade SHoot'em'Up Games

Requirements :
--------------
SDL2, SDL2_image, SDL2_ttf :

```bash
# On OSX:
brew install -v sdl2 sdl2_image sdl2_ttf
# On Linux Debian-ish distros (not tested)
apt-get install sdl2 sdl2_image sdl2_ttf
```

Building :
----------
```bash
make
```

Launching :
-----------
```bash
./bin/mashug-dbg
```
